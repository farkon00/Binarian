# Arithmetical operations
  ## add
  Return type : list
  
  Adds two numbers in list format. Returns sum of them in list format.

  Arguments :
  * num1 : list
  * num2 : list

  ## subt
  Return type : list
  
  Subtracts two numbers in list format. Returns difference of them in list format.

  Arguments :
  * num1 : list
  * num2 : list
